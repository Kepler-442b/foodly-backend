## 상품 관련 테이블

&nbsp;
### categories
- 상품 카테고리 정보를 저장한다.
- 상품 테이블과 ManyToMany관계를 가진다. 

&nbsp;
### products
- 상품의 정보를 저장한다.
- 카테고리와 ManyToMany 관계를 가진다.

&nbsp;
### products_categories
- 상품과 카테고리의 관계정보를 저장합니다.
- 상품과 카테고리는 외부키를 가져옵니다.
